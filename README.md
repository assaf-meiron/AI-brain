# AI-Brain

The central knowledge hub for **CheckIT** and **FollowIT**.

AI-Brain contains everything product teams need to know: product strategies, roadmaps, feature documentation, user research, and AI-powered tools to accelerate product development.

## For Product Managers

- 📋 **PRD Templates** — Ready-to-use templates and examples
- 🎯 **Strategies & Roadmaps** — Clear product direction and priorities
- 👥 **User Research** — Personas, needs, and insights
- 🤖 **AI Skills** — Tools to write better PRDs, faster

## For R&D Teams

- 🗺️ **Feature Context** — Understand the "why" behind features
- 📖 **Product Strategy** — Strategic decisions and positioning
- 🎯 **Roadmap** — See what's coming and timeline

## For Executives

- 📊 **Product Overviews** — Vision, mission, and key metrics
- 🛣️ **Roadmaps** — Feature timeline and priorities
- 🎯 **Strategy** — Market positioning and competitive advantages

## Quick Start

1. **New to AI-Brain?** Read [CLAUDE.md](CLAUDE.md) for the complete guide
2. **Looking for a product?** Go to `products/CheckIT/` or `products/FollowIT/`
3. **Need to write a PRD?** Check `prd-templates/` and use our AI skills

## What's Inside

```
products/          → Product-specific strategies, roadmaps, and features
knowledge-base/    → User personas, stakeholder needs, and research
prd-templates/     → Ready-to-use PRD templates and examples
skills/            → AI-powered tools for product teams
agents/            → AI agents for research and analysis
```
